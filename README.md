\# Vietnam Job Market Analytics



Ứng dụng phân tích thị trường việc làm Việt Nam \& Dự đoán mức lương bằng Machine Learning.



\## Tính năng chính

\- Phân tích dữ liệu khám phá (EDA)

\- Dự đoán mức lương theo thành phố, cấp bậc, kinh nghiệm

\- Giao diện web bằng Streamlit



\## Công nghệ sử dụng

\- Python + Streamlit

\- Pandas, Plotly

\- Scikit-learn (Random Forest Regressor)



\## Hướng dẫn chạy project



```bash

\# 1. Cài đặt thư viện

pip install -r requirements.txt



\# 2. Train model (chạy một lần)

python train\_model.py



\# 3. Chạy ứng dụng

streamlit run app.py
